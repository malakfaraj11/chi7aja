# 🛒 MERN Marketplace - Plateforme d'Achat et Vente

Une application web complète de commerce électronique de type marketplace, permettant aux utilisateurs de lister des produits, de gérer leur panier et d'effectuer des transactions sécurisées.

---

## 🚀 Aperçu du Projet

Ce projet est une application Full-Stack utilisant le stack **MERN**. Il sépare clairement le **Frontend** (React) du **Backend** (Node/Express) pour une architecture moderne et évolutive.



## 🛠️ Technologies Utilisées

* **Frontend :** React.js, Redux (Gestion d'état), Tailwind CSS (Styling), Axios.
* **Backend :** Node.js, Express.js.
* **Base de données :** MongoDB avec Mongoose.
* **Authentification :** JSON Web Tokens (JWT) & Bcrypt.js.
* **Paiement :** Intégration de l'API Stripe.
* **Hébergement :** Vercel (Frontend) / Render ou Heroku (Backend).

## 📋 Fonctionnalités (User Stories)

### 👤 Utilisateurs & Profils
- Création de compte et connexion sécurisée.
- Gestion du profil (adresse de livraison, historique).
- Rôles : Utilisateur standard et Administrateur.

### 📦 Gestion des Produits (Vendeurs)
- Publication d'annonces avec images.
- Gestion des stocks et modification des prix.
- Suppression d'articles obsolètes.

### 🛍️ Expérience Client
- Recherche multicritères et filtres par catégories.
- Panier d'achat persistant.
- Processus de paiement (Checkout) sécurisé.
